### 1. Patient Management Module (CRUD)
* **Registration**: Register new patients with details like medical condition and blood type.
* **Update**: Edit patient details (e.g., change of address or medical status) without deleting records.
* **View**: Dashboard view to list all active patients.
* **Delete**: Remove patient records from the system.

### 2. Appointment System
* **Book Appointments**: Patients can book slots with specific doctors (General, Surgeon, Pediatrician).
* **Manage Bookings**: View status and cancel upcoming appointments.

### 3. User Flow & Security
* **Landing Page**: Clear separation for "New Users" (Registration) and "Existing Patients" (Dashboard).
* **Persistent Database**: Utilizes **H2 Database (File-based)** to ensure data is saved permanently and survives server restarts.
